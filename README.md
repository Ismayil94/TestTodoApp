# Gelato task

## Intro
The repository contains 2 directories:
1. cypress_test
2. postman_test

**cypress_test** uses `cypress/included:7.2.0` image which has all the dependecies of OS and browsers to run the task. 

**postman_test** uses `postman/newman:5-alpine` image.

## How to Run

1. cypress_test:

```bash
$ docker-compose up
```

2. postman_test
```bash
$ docker-compose up
```
